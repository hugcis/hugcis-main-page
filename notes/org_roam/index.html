Org-roam

Org-roam is an org-mode package that implements features similar to Roam research:

  • Bi-directional links between notes — a link from a note to another note is also a “backlink” from the latter to the former.
  • Possibility to reference whole notes or subtrees transparently
  • Citation and references as links
  • And many others
